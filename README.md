# Calomong (칼로몽) 🌿

**칼로리(Calorie) + 꿈(Mong)**  
웹 기반 건강 관리 플랫폼 **Calomong**은 식단과 운동을 손쉽게 기록하고,  
챌린지와 랭킹 시스템을 통해 건강한 삶을 재미있게 관리할 수 있도록 도와주는 서비스입니다.

---

## 📌 주요 기능

- ✅ **식단 및 운동 기록 (CRUD)**
- 📊 **칼로리 섭취/소모 시각화 (Recharts, Chart.js)**
- 🏆 **챌린지 참여 및 랭킹 시스템** (게임화 요소 도입)
- 💬 **커뮤니티 게시판 (글쓰기, 댓글, 수정/삭제)**
- 🌗 **다크모드 지원**
- 👤 **로그인 / 회원가입 / 내 정보 수정**

---
